# Fixture Register (template)

> This file is a static skeleton. The **authoritative** register is
> generated by:
>
> ```bash
> cd arie-backend
> python -m fixtures.cli register --out fixtures/REGISTER.md
> ```
>
> Re-generate after every change to `registry.py`. Commit the
> generated `REGISTER.md` so reviewers can see what would be seeded
> without running the seeder.
