## Missing / Inconsistent Mappings Found

- Generic Enhanced Requirement uploads previously used `doc_type='enhanced_requirement'`; fixed to canonical doc types.
- Enhanced Requirement uploads previously did not queue Agent 1 verification; fixed for active runtime policies.
- EDD manual-review-only evidence remains intentionally non-automated.
- Tax/payslip/inheritance/sale/loan/adverse-media/senior-approval EDD families are not current default requestable documents and should not be represented as active automated runtime verified.
